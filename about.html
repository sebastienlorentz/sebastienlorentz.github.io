Tools

I reach for MATLAB for numerical work, Python for scripting and data, and p5.js for anything that benefits from an interactive canvas. Mathematics on this site is rendered with MathJax 3 — inline expressions like \(f(z) = z^2\) and display equations like

\[ \oint_\gamma f(z)\,\mathrm{d}z = 2\pi i \sum_k \operatorname{Res}(f,\, z_k) \]

render correctly in any browser without any server-side processing.
