Overview

A Voronoi diagram partitions the plane into regions based on proximity to a set of seed points. Each region \(V_i\) consists of all points closer to seed \(\mathbf{s}_i\) than to any other seed:

\[ V_i = \bigl\{\, \mathbf{x} \in \mathbb{R}^2 : \|\mathbf{x} - \mathbf{s}_i\| \leq \|\mathbf{x} - \mathbf{s}_j\| \text{ for all } j \neq i \,\bigr\} \]

This implementation takes a brute-force per-pixel approach: for every pixel \((x, y)\) on the canvas, we compute the Euclidean distance to each seed and assign the pixel to the nearest one.

Implementation notes

Brute-force complexity

For \(N\) seeds and a canvas of \(W \times H\) pixels, the naive approach runs in \(O(N \cdot W \cdot H)\) time. For this sketch (\(N = 22\), \(480 \times 340\)) that is around 3.6 million distance computations — fast enough to run in under a second, but the approach does not scale. A sweep-line implementation of Fortune's algorithm would reduce construction to \(O(N \log N)\).

Edge detection

Rather than tracing boundaries geometrically, we detect edges per-pixel. A pixel at \((x, y)\) is classified as an edge pixel when the gap between its nearest and second-nearest seed distances falls below a threshold \(\varepsilon\):

\[ \text{edge}(x,y) \iff d_2(x,y) - d_1(x,y) < \varepsilon \]

where \(d_1\) and \(d_2\) are the nearest and second-nearest Euclidean distances respectively. The value \(\varepsilon = 2.5\,\text{px}\) gives clean, single-pixel-wide boundaries without geometric edge-tracing.

Pixel density

When writing directly into p5's pixel buffer via loadPixels(), it is essential to call pixelDensity(1) in setup(). On retina / HiDPI displays the default pixel density is 2, making the buffer four times larger than expected; index arithmetic then silently corrupts the image.

Source

The full annotated source is on GitHub. The core loop:

for (let y = 0; y < H; y++) {
  for (let x = 0; x < W; x++) {
    let minD = Infinity, secD = Infinity, nearest = 0;
    for (let i = 0; i < seeds.length; i++) {
      let d = (x - seeds[i].x)**2 + (y - seeds[i].y)**2;
      if (d < minD) { secD = minD; minD = d; nearest = i; }
      else if (d < secD) { secD = d; }
    }
    // colour pixel according to nearest seed ...
  }
}
